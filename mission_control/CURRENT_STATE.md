## 현재 구현과 목표의 주요 차이

| 영역 | 현재 구현 | 목표 |
|---|---|---|
| 브랜드 | BarBot | Re:Station |
| 캐릭터 | 이름 없는 단일 바텐더 | 카루아 |
| 대화 스타일 | 범용 위로와 규칙 응답 | 반존대, 능청, 뼈 있는 농담, 환기 |
| 추천 모델 | `Cocktail`과 `CocktailRecord` 이중 구조 | 단일 데이터 모델 |
| 추천 입력 | 단맛, 신맛, 도수, 탄산 중심 | 기분, 상황, 맛 취향, 무알코올, 제외 재료 |
| 추천 출력 | 카드와 일반 대사 | 구조화된 추천 이유와 카루아식 한마디 포함 |
| 생성형 대화 | 미연결 OpenAI 및 Ollama 모듈 | Web Worker 기반 WebLLM |
| UI 색상 | 갈색 및 골드 중심 | 어두운 남색 또는 흑회색과 연보라 |
| 테스트 | 없음 | 추천, 저장, 주요 흐름 자동 테스트 |
