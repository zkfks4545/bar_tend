# WebLLM 후보 잠정 순위

## 결론

공개 자료와 설치된 WebLLM 카탈로그를 기준으로 후보 순위만 잠정 확정한다. 최종 제품 모델은 아직 결정하지 않는다.

| 잠정 순위 | 후보 | 판단 |
|---:|---|---|
| 1 | `Qwen3.5-2B-q4f16_1-MLC` | 품질 여유와 브라우저 부담 사이의 가장 균형 잡힌 우선 후보 |
| 2 | `Qwen3.5-4B-q4f16_1-MLC` | 더 큰 품질 여유를 기대할 수 있으나 요구 VRAM 부담이 큼 |
| 3 | `Qwen3.5-0.8B-q4f16_1-MLC` | 접근성은 좋지만 캐릭터 말투와 복합 지시 유지 위험이 큼 |
| 4 | `Qwen3-1.7B-q4f16_1-MLC` | 요구 VRAM이 1순위와 비슷해 이전 세대 선택의 이점이 작음 |
| 5 | `gemma3-1b-it-q4f16_1-MLC` | 매우 가볍지만 현재 프로젝트의 Qwen 우선 방향과 거리가 있음 |

이 순위는 모델별 로컬 다운로드·실행 비교 결과가 아니다. 공개 데이터만으로 정한 구현 우선순위이며, 최종 모델 확정이나 다른 후보 배제를 의미하지 않는다.

## 확인된 사실

프로젝트에 고정된 `@mlc-ai/web-llm@0.2.84`의 `prebuiltAppConfig`를 기준으로 확인했다.

| 모델 ID | 카탈로그 등록 | 요구 VRAM |
|---|---:|---:|
| `Qwen3.5-2B-q4f16_1-MLC` | 예 | 2245.44 MB |
| `Qwen3.5-4B-q4f16_1-MLC` | 예 | 3867.82 MB |
| `Qwen3.5-0.8B-q4f16_1-MLC` | 예 | 1629.49 MB |
| `Qwen3-1.7B-q4f16_1-MLC` | 예 | 2036.66 MB |
| `gemma3-1b-it-q4f16_1-MLC` | 예 | 711.07 MB |

WebLLM은 브라우저 안에서 WebGPU를 이용해 모델을 실행하며, 실제 속도와 성공 여부는 사용자 기기와 브라우저에 따라 달라진다.

## 순위 기준

1. 한국어 대화와 캐릭터 말투를 유지할 것으로 기대되는 품질 여유
2. 설치된 WebLLM 버전의 직접 지원 여부
3. 브라우저 요구 VRAM과 다운로드 부담
4. 짧은 응답과 구조화 지시를 처리할 여지
5. 프로젝트가 우선 후보로 정한 Qwen 계열과의 부합성

공개 자료만으로 카루아 말투 적합성이나 실제 한국어 품질을 증명할 수 없으므로, 확인되지 않은 벤치마크 점수와 예상 속도는 순위 근거로 사용하지 않는다.

## 최종 확정 조건

최종 모델은 아래 항목을 검토한 뒤 별도 의사결정으로 확정한다.

- 제품이 허용할 다운로드 크기와 최소 목표 기기
- 프롬프트 설계 후 필요한 모델 품질 수준
- 카루아·시에스타 RP 평가 계약 충족 가능성
- WebGPU 미지원 및 모델 실패 시 규칙 엔진 복구 경로
- 실제 배포 전 모델 카드의 라이선스와 배포 조건

후보를 하나씩 로컬에서 내려받아 비교하는 작업은 계획하지 않는다. 구현이 필요하면 잠정 1순위부터 시작할 수 있지만, 이는 최종 선정으로 기록하지 않는다.

## 근거 수준과 한계

| 구분 | 상태 |
|---|---|
| WebLLM 지원 모델 ID | 설치 패키지에서 확인 |
| 요구 VRAM | 설치 패키지에서 확인 |
| 후보 순위 | 공개 자료 기반 잠정 판단 |
| 최종 제품 모델 | 미정 |
| 한국어 자연스러움 | 실측하지 않음 |
| 카루아·시에스타 RP 적합성 | 실측하지 않음 |
| 다운로드 시간·TTFT·Tok/s | 실측하지 않음 |
| 라이선스 및 배포 조건 | 실제 배포 전 모델 카드 기준으로 재확인 필요 |
